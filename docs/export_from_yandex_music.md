# Как вытащить список лайкнутых треков из Яндекс Музыки

У Яндекс Музыки нет удобной штатной кнопки «экспортировать мои лайки в TXT».  
Поэтому самый простой рабочий способ — открыть список в браузере и собрать названия с уже загруженной страницы.

Скрипт из этого репозитория ничего никуда не отправляет. Он только смотрит на открытую страницу, собирает видимые треки и скачивает TXT-файл на ваш компьютер.

## 1. Откройте нужную страницу

В Яндекс Музыке:

```text
Коллекция → Мне нравится → Треки
```

Нужна именно вкладка `Треки`.

## 2. Откройте консоль браузера

Firefox:

```text
Ctrl + Shift + K
```

Chrome / Edge:

```text
F12 → Console
```

Если браузер предупреждает об опасности вставки кода, введите:

```text
allow pasting
```

Это стандартная защита браузера от вставки случайного кода. Вставляйте только тот код, который понимаете и которому доверяете.

## 3. Запустите сборщик

Откройте файл:

```text
tools/yandex_music_export_console.js
```

Скопируйте его целиком в консоль и нажмите `Enter`.

В консоли появится сообщение:

```text
Сборщик запущен. Листайте список до конца, потом выполните ymSave()
```

## 4. Пролистайте список до конца

Это обязательный шаг.

Яндекс Музыка не держит все треки на странице сразу. Она подгружает их по мере прокрутки.  
Скрипт собирает только то, что уже появлялось на странице.

Можно листать колесом мыши, клавишей `Page Down` или протяжкой ползунка.

## 5. Сохраните файл

Когда дошли до конца списка, выполните в консоли:

```js
ymSave()
```

Должен скачаться файл:

```text
yandex_liked_tracks.txt
```

Формат строк:

```text
Artist - Track
Artist - Track
Artist - Track
```

## 6. Проверьте файл

Откройте файл любым текстовым редактором и быстро проверьте:

- нет ли строк `Unknown artist`;
- нет ли мусора из интерфейса;
- треков примерно столько, сколько ожидали;
- строки выглядят как `Исполнитель - Название`.

Если треков мало — значит страницу не долистали до конца. Откройте Яндекс Музыку, запустите сборщик ещё раз и долистайте список полностью.

## 7. Что делать дальше

Положите `yandex_liked_tracks.txt` рядом со скриптом:

```text
spotify_import_yandex_liked.py
```

И запускайте импорт в Spotify по инструкции из `README.md`.
