# Changelog

## 0.1.0

Первый публичный вариант.

- экспорт списка треков из открытой страницы Яндекс Музыки через браузерную консоль;
- импорт TXT-списка в Spotify-плейлист;
- сохранение состояния после остановки или rate limit;
- отчёты `matches.csv`, `needs_review.csv`, `not_found.csv`;
- пример входного файла;
- `.gitignore` для личных файлов, кэшей и токенов.
